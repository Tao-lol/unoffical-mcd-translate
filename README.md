# unoffical-mcd-translate
这里是Minecraft Dungeons非官方中文语言包的存档项目，仅用于存档，不接受任何issue或pull requests。

使用方法：  
打开游戏安装目录（也就是启动器目录）  
进入 \products\dungeons\dungeons\Dungeons\Content\Paks ，将Pak文件直接复制到此处即可  
注意：请不要重命名文件
