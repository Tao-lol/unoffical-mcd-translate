# unoffical-mcd-translate
这是Minecraft Dungeons非官方中文语言包的存档项目，仅用于存档，不接受任何issue或pull requests。

注意：此语言包基于英语制作，会覆盖语言选项内的英语，卸载语言包即可恢复！  

使用方法：  
打开游戏安装目录（也就是启动器目录）  
进入 \products\dungeons\dungeons\Dungeons\Content\Paks ，将Pak文件直接复制到此处即可  
卸载：删除Pak文件即可  
注意：请不要重命名文件
